## First-Time Setup

After installation:

1. Open CuddleKine.
2. Go to Settings.
3. Configure a model provider.
4. Configure Tencent Cloud COS if you want reference-image generation with cloud models.
5. Create a test order and generate one sample.

Guides:

- [Install CuddleKine](../setup/install-cuddlekine.md)
- [Configure model providers](../setup/model-providers.md)
- [Configure Tencent Cloud COS](../setup/tencent-cos.md)
- [Common issues](../setup/troubleshooting.md)

## Known Notes

- API keys are stored locally and are not included in the installer.
- Local ComfyUI quality depends on the user's own ComfyUI models and workflows.
- Cloud providers may charge per generation.
