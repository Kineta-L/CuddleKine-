# CuddleKine

CuddleKine is an AI-assisted plush toy sampling workbench for turning product briefs and reference images into manufacturable plush toy concepts.

The application lives in [`CuddleKine/`](./CuddleKine/).

## Highlights

- Desktop workbench built with React, Vite, and Tauri.
- FastAPI backend for orders, briefs, generation records, exports, and provider settings.
- Multi-provider image generation support for ComfyUI, OpenAI, Replicate, and mock mode.
- Prompt system focused on plush toy feasibility, reference fidelity, clean product presentation, and full-subject preservation.
- Local-first workflow with generated data, outputs, and API keys excluded from Git.
